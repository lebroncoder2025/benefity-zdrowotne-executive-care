\question{1}{Model finansowania -- abonament vs fee for service}{%
To punkt, od kt\'orego warto zacz\k{a}\'c.\\[0.25em]
Abonament = plac\'owka dosta\l{}a pieni\k{a}dze z g\'ory $\rightarrow$ termin\'ow brakuje.\\
Fee for service = p\l{}atno\'s\'c za realn\k{a} wizyt\k{e} $\rightarrow$ specjalista jest dost\k{e}pny.\\
Dost\k{e}pno\'s\'c i jako\'s\'c opieki zaczyna si\k{e} w\l{}a\'snie tutaj.
}

\question{2}{Czy mo\.zesz obj\k{a}\'c pakietem rodzic\'ow?}{%
Ma\l{}.zonek i dzieci w polisie to dzi\'s niemal standard. Ale mo\.zliwo\'s\'c obj\k{e}cia rodzic\'ow to benefit,
kt\'ory naprawd\k{e} robi efekt WOW i wyr\'o\.znia pracodawc\k{e} na tle rynku.
}

\question{3}{Czy wiesz, czego potrzebuj\k{a} Twoi pracownicy?}{%
Pakiety ,,z katalogu'' rzadko trafiaj\k{a} w sedno. Dlatego pierwszym krokiem powinna by\'c ankieta potrzeb.\\[0.35em]
{\color{branddark}\bfseries Wskaz\'owka:} Mamy gotowe, sprawdzone narz\k{e}dzie. Chcesz to zrobi\'c dobrze? Odezwij si\k{e}.
}

\question{4}{Ile realnie czeka si\k{e} na wizyt\k{e}?}{%
Regulaminy swoje, \.zycie swoje. Najprostszy test: sprawd\'z, ile czeka pacjent z pakietem, a ile prywatny klient.\\[0.35em]
{\color{branddark}\bfseries Wskaz\'owka:} Mamy sprawdzony spos\'ob, jak weryfikowa\'c dost\k{e}pno\'s\'c w praktyce.
}

\question{5}{Kto faktycznie p\l{}aci za pakiet?}{%
Nawet niewielkie dop\l{}aty pracownika mog\k{a} skutkowa\'c rezygnacj\k{a}. Dane rynkowe pokazuj\k{a} jasno:
finansowanie przez firm\k{e} zwi\k{e}ksza realne korzystanie z pakietu.
}

\question{6}{Czy po podpisaniu umowy kto\'s si\k{e} zajmie Twoj\k{a} firm\k{a}?}{%
Agent, broker, support czy tylko infolinia? To decyduje, czy HR b\k{e}dzie mia\l{} wsparcie,
czy zostanie sam z pytaniami pracownik\'ow.
}
