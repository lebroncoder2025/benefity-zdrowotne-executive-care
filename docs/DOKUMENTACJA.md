# Dokumentacja wariantu Executive Care

## Charakter wariantu

Executive Care to spokojna, premium wersja serwisu. Buduje zaufanie przez porządek informacji, elegancką typografię i ton bardziej doradczy niż promocyjny.

## Główne cele

- przesunięcie komunikacji z poziomu „benefitu” na poziom stabilności biznesu
- mocniejsze pokazanie wiarygodności i dojrzałości marki
- skrócenie ścieżki do kontaktu bez agresywnej sprzedaży
- uporządkowanie strony głównej, podstrony „O nas” oraz sekcji wydarzeń HR

## Struktura informacji

- `index.html`: hero, przewagi, korzyści biznesowe, proces, moduł oferty specjalnej, kontakt
- `o-nas.html`: pozycjonowanie zespołu, wyniki, kompetencje i zasady współpracy
- `tam-gdzie-dzieje-sie-hr.html`: wydarzenia HR jako element reputacji marki

## Decyzje wizualne

- fonty: `Fraunces` + `Manrope`
- paleta: ciemna zieleń, piaskowy beż, ciepłe światło premium
- układ: duże oddechy, miękkie karty, spokojne tempo sekcji
- styl zdjęć: portrety i fotografie biznesowe potraktowane jako element zaufania

## Mocne strony tego kierunku

- najlepiej wspiera sprzedaż relacyjną i marki premium
- dobrze wygląda w rozmowie z właścicielem firmy i kadrą zarządzającą
- daje wrażenie spokoju, jakości i przewidywalności
- nadaje się do dalszej rozbudowy o formularz leadowy lub FAQ

## Technicznie

- statyczny frontend: HTML + CSS + JavaScript
- gotowe do publikacji przez GitHub Pages
- brak zależności od WordPressa po wdrożeniu tego wariantu
- obecne kanały kontaktu zachowane: telefon, e-mail, WhatsApp, Messenger
